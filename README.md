# Love App - Danh Sách Ước Mơ 🌹

Ứng dụng giúp bạn và người ấy cùng nhau xây dựng danh sách những món quà mong ước, được đồng bộ thời gian thực qua Firebase.

## Hướng dẫn triển khai (Deployment)

Để đưa ứng dụng lên Internet và chia sẻ cho người khác, bạn có thể tham khảo [Hướng dẫn triển khai chi tiết](./deployment_guide.md) (hoặc xem file trong thư mục artifacts).

### Các bước nhanh:
1. Đưa code lên **GitHub**.
2. Kết nối với **Vercel** hoặc **Netlify**.
3. Cấu hình **Firebase Rules** để cho phép đọc/ghi.

Chúc bạn có những giây phút ngọt ngào! ❤️
